# Cauldron — Releases

Public download and auto-update feed for **Cauldron**, the local, chat-driven
structural-biology workbench by Chesseli.

## Download

Get the latest installer from the [**Releases**](https://github.com/chesseli/cauldron-releases/releases/latest) page:

| OS | File | Install |
|----|------|---------|
| **Windows** | `Cauldron-Setup-<version>.exe` | Double-click the installer. |
| **Linux (Debian/Ubuntu)** | `cauldron_<version>_amd64.deb` | `sudo apt install ./cauldron_<version>_amd64.deb` |
| **Linux (portable)** | `Cauldron-<version>.AppImage` | `chmod +x` and run — no install. |

Installed apps auto-update from this repository (AppImage + Windows).

This repo holds **build artifacts only** — the application source lives elsewhere.
